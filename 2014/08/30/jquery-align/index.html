简介

一个使用jQuery元素定位插件，用于将元素快速进行定位，目前可以将如下内容作为基准定位：

  • 目标元素
  • 当前屏幕
  • 整个页面
  • 鼠标事件
  • 坐标

使用

在使用前必须引入jQuery

配置

base

默认为screen

jQuery选择器或节点

当前屏幕

整个页面

鼠标事件

页面相对坐标

points

含有两个字符串的数组，前者为参照元素的基准点名称，后者为被定位元素的基准点名称，这两个基准点将重合。

默认为['cc', 'cc']

基准点名称如下图所示：

offset

含有两个数字的数组，前者为x轴偏移，后者为y轴偏移，默认为[0, 0]

fixed

仅当base为screen时启用，表示是否固定与屏幕，默认为false
