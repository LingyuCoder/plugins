简介

一个使用jQuery元素定位插件，用于将元素快速进行定位，目前可以将如下内容作为基准定位：

  • 目标元素
  • 当前屏幕
  • 整个页面
  • 鼠标事件
  • 坐标

依赖

  • jQuery

使用

配置

base

元素位置的基准元素，提供如下类型，默认为'screen'

jQuery选择器或节点

我会根据蓝方块来定位，我俩中心会重合

当前屏幕

显示在屏幕正中间，但不随着页面滚动而移动

整个页面

显示在页头右上角

将显示在页面右上角

鼠标事件

点击空白处，我将飞过去

页面相对坐标

显示坐标为x:400 y:3300

基准点名称

含有两个字符串的数组，前者为参照元素的基准点名称，后者为被定位元素的基准点名称，这两个基准点将重合。默认为:['cc', 'cc']

基准点名称如下图所示：

我的左上角将和蓝方块的中心重合

offset

含有两个数字的数组，前者为x轴偏移，后者为y轴偏移，默认为: [0, 0]

我的左上角，将在蓝方块的左上角右下10px的位置

fixed

仅当base为screen时有效，表示是否固定与屏幕，默认为false

我会一直显示在屏幕中间，记得点关闭
