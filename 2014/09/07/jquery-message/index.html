简介

实例

配置

content

自定义展示的内容，为可以为HTML字符串

duration

消息展示的时间，默认为5000，如果为false，则不会自动关闭

closeWhenClick

展示的消息能否通过点击来关闭，默认为true

css

消息的自定义样式，默认为'wd-ui-msg wd-ui-log'，样式来自jquery-message.css
